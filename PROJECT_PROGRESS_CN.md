# 网站项目进展看板

## 项目概览

### 项目名称

个人网站重构与上线项目

### 项目目标

建设一个面向中国大陆访问的个人网站，逐步完成以下能力：

- 个人博客发布
- 算法学习专区
- 每日打卡与学习记录
- 笔记上传、代码上传与批注
- 自建服务器部署与后续可持续迭代

### 当前工作区

`F:\website`

### 当前总状态

- 当前阶段：前端首版完成 + 后端基础 API 原型阶段
- 当前重点：补齐前端 V1 细节、把后端数据从内存切到持久化、解决前端生产构建问题
- 部署方向：自建服务器 + 域名 + ICP 备案 + Caddy
- 目标访问区域：中国大陆
