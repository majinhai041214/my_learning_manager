# 网站项目进展看板

## 项目概览

### 项目名称

个人网站重构与上线项目

### 项目目标

建设一个面向中国大陆访问的个人网站，逐步完成以下能力：

- 个人博客发布
- 算法学习专区
- 每日打卡与学习记录
- 学习笔记上传、分类、预览与批注
- 可编辑学习路线图
- 自建服务器部署与后续持续迭代

### 当前工作区

`F:\website`

### 当前总状态

- 当前阶段：前端核心功能版已完成 + 后端文件持久化已接入
- 当前重点：收尾博客内容体系、稳定前端生产构建、为云服务器部署做准备
- 部署方向：自建服务器 + 域名 + ICP 备案 + Caddy
- 目标访问区域：中国大陆
